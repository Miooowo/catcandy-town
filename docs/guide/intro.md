# 游戏介绍

欢迎来到 **猫果镇物语**！

## 这是什么？
这是一款**挂机放置类模拟游戏**。你不需要主动管理或操作，而是以**旁观者的视角**观察小镇居民的日常生活。游戏会自动运行，居民们会自主行动，你可以随时查看他们的工作、社交、恋爱、结婚等有趣的故事。

## 游戏类型
- 🎮 **挂机放置**：游戏自动运行，无需频繁操作
- 👁️ **旁观者视角**：你只是观察者，不直接干预居民的生活
- ⏱️ **实时模拟**：居民们会根据自己的性格和情况自主做出决策

## 核心机制
1. **自主行动**：每个居民都有独特的性格和特性，他们会根据自己的情况自主选择行动，包括工作、社交、恋爱、建设等。
2. **建筑系统**：居民可以参与建设各种建筑，包括公园、酒吧、酒店、教堂、电影院等。建筑建成后会自动运营。
3. **选举系统**：当建筑需要工作人员时，居民会通过投票选举管理者，管理者可以招聘下属员工。
4. **关系系统**：居民之间会自然发展友谊、恋爱、婚姻等关系，甚至可能发生出轨、离婚等戏剧性事件。

## 游戏特色
- 🎭 **丰富的性格系统**：每个居民都有随机生成的性格（如易怒、开朗、温柔等）和特性（如喜欢睡觉、爱钱等）
- 🏗️ **建筑系统**：多种建筑类型，每种都有不同的功能和营业时间
- 💼 **工作系统**：居民可以应聘工作，通过选举或招聘获得职位，获得工资收入
- ❤️ **关系系统**：居民之间会发展复杂的人际关系，包括朋友、恋人、夫妻等
- 💰 **经济系统**：居民通过工作、建设等方式获得收入，建筑也会产生收入
- 📊 **数据统计**：可以查看建筑收入、工作人员工资等详细数据
- 🌙 **暗色模式**：支持亮色/暗色主题切换
- 💾 **存档系统**：支持自动存档、手动存档、导出/导入存档

## 如何游玩
1. 点击"开始游戏"按钮启动游戏
2. 观察居民们的自主行动和互动
3. 可以点击居民卡片查看详细信息
4. 可以点击建筑卡片查看建筑详情和收入统计
5. 可以调整游戏时间速度
6. 可以随时暂停、保存或重置游戏

## 技术栈
本游戏使用以下技术构建：
- Vue 3（响应式框架）
- TypeScript（类型安全）
- VitePress（文档和游戏平台）
